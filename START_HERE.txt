3-STEP QUICK START
═════════════════════════════════════════════════════════════════════════════

STEP 1: Open Terminal 1
  cd backend
  docker-compose up --build
  
  (Wait until all 5 services say "STARTED" - takes 1-2 minutes)

STEP 2: Open Terminal 2
  npm install
  npm run dev
  
  (Wait for "Local: http://localhost:5173/")

STEP 3: Open Browser
  Go to http://localhost:5173
  Click "Sign In"
  Use demo account:
    Email: donor@demo.com
    Password: Demo@123

WHAT HAPPENS WHEN YOU RUN IT
═════════════════════════════════════════════════════════════════════════════

Step 1 (docker-compose up):
  • Docker downloads images (if not cached)
  • Builds 5 Java microservices
  • Starts PostgreSQL database
  • Applies database schema automatically
  • All services start and become ready

Step 2 (npm install && npm run dev):
  • Downloads React dependencies
  • Starts Vite development server
  • Frontend can now make API calls to backend

Step 3 (Browser):
  • Vite hot-reloads with live updates
  • Login flow connects to backend
  • All data comes from real PostgreSQL database
  • Full end-to-end system working

VERIFY EVERYTHING IS WORKING
═════════════════════════════════════════════════════════════════════════════

After startup, verify by checking:

1. Backend is running:
   curl http://localhost:8080/actuator/health
   Should return: {"status":"UP"}

2. Frontend is running:
   Visit: http://localhost:5173
   Should show homepage

3. Connection is working:
   Login and check DevTools (F12)
   Network tab should show API calls to localhost:8080

4. Database is connected:
   Check that login works and data loads
