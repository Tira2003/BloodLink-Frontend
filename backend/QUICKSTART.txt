🚀 QUICK START (5 MINUTES)
==========================

1. Navigate to backend directory:
   cd backend

2. Start all services:
   docker-compose up --build

   (Or use: ./docker-compose-up.sh)

3. Wait for services to start (2-3 minutes)

4. Verify all services are running:
   curl http://localhost:8080/actuator/health

5. Test the API:
   curl -X POST http://localhost:8080/api/users/register \
     -H "Content-Type: application/json" \
     -d '{"email":"test@example.com","password":"Test123","firstName":"Test","lastName":"User","role":"DONOR"}'

✅ SUCCESS! All services are running.

🆘 TROUBLESHOOTING
==================

Services won't start?
  → Check Docker is running: docker ps
  → Clean Docker: docker-compose down -v
  → Rebuild: docker-compose up --build

Database connection error?
  → Check PostgreSQL running: psql -h localhost -U postgres
  → Check credentials in application.yml
  → Restart database: docker-compose restart postgres

Port already in use?
  → Kill process: lsof -i :PORT | grep LISTEN | awk '{print $2}' | xargs kill -9
  → Or change port in docker-compose.yml

See SETUP_GUIDE.md for detailed troubleshooting!
